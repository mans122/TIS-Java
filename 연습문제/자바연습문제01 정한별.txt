2. java에서 코드작성부터 실행까지의 순서를 c언어와
=>C언어는 코드를 작성하고 컴파일러/링커가 .exe파일로 생성해준다.
=>Java는 코드를 작성하고 컴파일러가 바이트코드로 된 .class파일이 생성되며 인터프리터 방식으로 JRE가 설치된 환경에서 JVM을 통해 실행된다.

3. 바이트코드에 대해서 설명하시오.
=>JVM에서만 실행되는 기계어로, 어떤CPU와도 관계없는 binary code 이다.

4. 가상머신에 대해서 설명하시오.
=>자바 가상머신(JVM : Java Virtual Machine)은 S/W로서 서로 다른 플랫폼에서 자바 프로그램이 실행될수 있게 동일한 환경을 제공하는 역할을 한다.

5. class와 instance가 무엇인지 설명하시오
=>class란 객체를 만들어내기 위한 설계도 혹은 틀,연관되어있는 변수와 Method의 집합
=>instance란 설계도를 바탕으로 S/W세계에 구현된 구체적인 실체,객체를 S/W에 실체화하면 그것을 'Instance라고 부른다.
  실체화된 Instance는 메모리에 할당된다.

6. main함수에서 static의 역할이 무엇인지 설명하시오.
=> C언어의 malloc 함수와 같이 해당 함수를 메모리에 할당시켜주는 역할을 한다.
